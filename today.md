# 오늘 작업 요약

## 개요
- MapleStory Worlds 매크로 로직 정리 및 동작 안정화
- 몬스터 감지(색상 기반) 추가/튜닝
- 3층( floor3 ) ↔ 3_2 이동/사냥 루프 구성
- 버프 타이머(F4 기준) 및 수동 입력 충돌 처리

## 주요 변경 사항
### 몬스터 감지
- 색상 기반 감지로 전환 (HSV 범위 사용)
- 감지 영역: `MONSTER_REGION = (620, 260, 1029, 462)`
- 로그에 위치 정보와 `Monster: O/X` 표시
- 감지 주기: 0.2초
- 저장/스크린샷 파일 생성 제거

### 핫키
- `F1`: 자동 움직임 재개 (버프 타이머 초기화 없음)
- `F2`: 일시정지
- `F3`: 게임 창 1280x720 리사이즈 + 포커스
- `F4`: 버프 즉시 사용 후 90초마다 자동 사용

### 버프(Q/E)
- `F4` 누를 때 즉시 사용 후 90초 간격 반복
- 몬스터가 없을 때만 사용, 몬스터가 있으면 대기 후 X가 되면 실행
- `Q` → 딜레이 → `E` 순서로 조정

### 수동 입력 충돌 대응
- `W` 수동 입력 감지 시 1초간 자동 동작 정지

### 3층/3_2 루프 로직
- `floor3`: 64 근처에서 사냥 후 일정 시간(변수: `FLOOR3_TO_3_2_DELAY_SEC`) 지나면 3_2 이동 시도
- `floor3_2`: 44 근처에서 10초 사냥 → 64±1로 이동 → 좌측 바라보기 고정 → 5초 사냥 → 몬스터 X 시 아래점프
- 이동 중 스킬 미사용
- 아래점프 실패 시 0.6초 간격으로 재시도
- `floor3_1`: `floor3_2`에서 내려온 경우에만 아래점프
- `iso_point`: 잠깐 정지 후 우측 점프로 복귀

## 중요 파라미터
- `FLOOR3_TO_3_2_DELAY_SEC`: 3층 사냥 시간 (현재 15초로 변경됨)
- `BUFF_INTERVAL_SEC`: 90초
- `MONSTER_COLOR_LOWER / UPPER`: 몬스터 색상 감지 범위 (다크 틸)
- `MONSTER_MIN_RATIO`, `MONSTER_MIN_PIXELS`: 감지 기준

## 테스트 메모
- 몬스터 감지는 O일 때 공격하도록 원복 완료
- 3_2 자리 고정 및 아래점프 재시도 동작 확인 중
